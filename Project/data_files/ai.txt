artificial intelligence=Field of computer science focused on building systems that perform tasks requiring human intelligence
intelligence=Ability to learn reason solve problems and adapt to new situations
agent=Entity that perceives environment and takes actions to achieve goals
environment=External context in which an agent operates
rational agent=Agent that selects actions maximizing expected performance
search=Process of exploring possible states to find a solution
state space=All possible configurations of a problem
heuristic=Rule of thumb guiding search toward optimal solutions
knowledge representation=Method of encoding information for intelligent reasoning
inference=Process of deriving new conclusions from known facts
machine learning=Subfield of ai enabling systems to learn from data
model=Mathematical representation learned from data
training=Process of teaching a model using data
testing=Process of evaluating model performance on unseen data
feature=Individual measurable property used as input to a model
label=Target output value in supervised learning
dataset=Collection of data used for training and evaluation
overfitting=Model performs well on training data but poorly on new data
underfitting=Model fails to capture underlying patterns in data
generalization=Ability of a model to perform well on unseen data
supervised learning=Learning approach using labeled data
unsupervised learning=Learning approach using unlabeled data
reinforcement learning=Learning through interaction using rewards and penalties
classification=Task of predicting discrete categories
regression=Task of predicting continuous values
clustering=Grouping similar data points together
dimensionality reduction=Reducing number of input features while preserving information
neural network=Computational model inspired by biological neurons
perceptron=Basic unit of a neural network performing linear classification
activation function=Function introducing nonlinearity into neural networks
backpropagation=Algorithm for training neural networks using error gradients
gradient descent=Optimization algorithm minimizing loss function
loss function=Measure of difference between predicted and actual values
epoch=One complete pass through the training dataset
batch=Subset of training data processed in one iteration
deep learning=Subset of machine learning using multi layer neural networks
convolutional network=Neural network specialized for image processing
recurrent network=Neural network designed for sequential data
transformer=Neural architecture using attention mechanisms for sequence modeling
attention=Mechanism allowing models to focus on important input parts
embedding=Vector representation of words or data in continuous space
natural language processing=Field enabling machines to understand human language
tokenization=Process of splitting text into smaller units
sentiment analysis=Task of identifying emotional tone in text
language model=Model predicting probability of word sequences
large language model=Massive language model trained on extensive text data
pretraining=Initial training on large general datasets
fine tuning=Additional training on specific tasks or domains
transfer learning=Using knowledge from one task to improve another
prompt=Input instruction given to a generative model
prompt engineering=Designing effective prompts to guide model output
generative ai=Ai systems capable of creating new content
generation=Process of producing new text images or data
text generation=Creating coherent textual content automatically
image generation=Creating visual content from textual descriptions
diffusion model=Generative model creating data by reversing noise process
gan=Generative adversarial network with generator and discriminator
generator=Model component that creates synthetic data
discriminator=Model component distinguishing real from generated data
multimodal=Model capable of processing multiple data types
hallucination=Model generating plausible but incorrect information
bias=Systematic error causing unfair or skewed outputs
ethics=Principles guiding responsible ai development and use
explainability=Ability to understand and interpret model decisions
interpretability=Degree to which humans can comprehend model behavior
alignment=Ensuring ai systems follow human values and intentions
safety=Measures preventing harmful behavior from ai systems
privacy=Protection of sensitive user data in ai systems
federated learning=Training models across decentralized devices without sharing raw data
edge ai=Running ai models on local devices instead of cloud
cloud computing=Remote servers providing scalable computation for ai
gpu=Specialized processor accelerating parallel computations
tpu=Hardware accelerator optimized for tensor operations
inference time=Time required for a model to produce predictions
latency=Delay between input and output in a system
throughput=Amount of data processed in a given time
scalability=Ability of a system to handle increased workload
evaluation metric=Standard used to measure model performance
accuracy=Proportion of correct predictions
precision=Proportion of true positives among predicted positives
recall=Proportion of true positives among actual positives
f1 score=Harmonic mean of precision and recall
confusion matrix=Table summarizing classification results
cross validation=Technique for assessing model performance using multiple splits
regularization=Technique reducing overfitting by adding penalty
dropout=Regularization method randomly disabling neurons during training
hyperparameter=Configurable setting controlling model training process
optimization=Process of improving model performance
fine grained control=Ability to adjust detailed aspects of model behavior
context window=Maximum input length a language model can process
temperature=Parameter controlling randomness in text generation
top k=Sampling method limiting choices to top probable tokens
top p=Sampling method selecting tokens within cumulative probability threshold
rlhf=Reinforcement learning from human feedback to refine model behavior
