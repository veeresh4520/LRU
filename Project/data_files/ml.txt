machine learning=Field of study enabling computers to learn patterns from data without explicit programming
artificial intelligence=Broader discipline focused on building intelligent systems
data=Raw facts used for training and evaluation
dataset=Structured collection of data samples
feature=Measurable input variable used by a model
label=Target output value for supervised tasks
sample=Single data instance in a dataset
model=Mathematical function mapping inputs to outputs
training=Process of learning model parameters from data
testing=Evaluating model performance on unseen data
validation=Intermediate dataset used for tuning models
inference=Using a trained model to make predictions
supervised learning=Learning from labeled data
unsupervised learning=Learning patterns from unlabeled data
semi supervised learning=Learning using small labeled and large unlabeled data
reinforcement learning=Learning by interacting with environment via rewards
classification=Predicting discrete categories
binary classification=Classification with two possible classes
multiclass classification=Classification with more than two classes
regression=Predicting continuous numerical values
clustering=Grouping similar data points together
dimensionality reduction=Reducing number of features while preserving structure
feature engineering=Process of selecting and transforming features
feature scaling=Normalizing feature ranges for better performance
normalization=Rescaling data to a standard range
standardization=Centering data around mean with unit variance
train test split=Dividing data into training and testing sets
cross validation=Evaluating model using multiple data splits
k fold=Cross validation method dividing data into k subsets
bias=Error due to overly simplistic model assumptions
variance=Error due to sensitivity to training data fluctuations
overfitting=Model memorizes training data and fails to generalize
underfitting=Model fails to capture underlying data patterns
generalization=Ability to perform well on new unseen data
loss function=Measure of prediction error
cost function=Average loss across training samples
optimization=Process of minimizing loss function
gradient descent=Iterative algorithm to minimize loss
learning rate=Step size in gradient descent updates
batch=Subset of data processed in one iteration
epoch=One full pass over the training dataset
stochastic gradient descent=Gradient descent using single samples
mini batch=Gradient descent using small data batches
regularization=Technique to prevent overfitting
l1 regularization=Penalty adding absolute weight values
l2 regularization=Penalty adding squared weight values
dropout=Technique randomly disabling neurons during training
early stopping=Halting training when validation error increases
hyperparameter=Configuration set before training begins
grid search=Systematic hyperparameter tuning method
random search=Hyperparameter tuning using random combinations
evaluation metric=Standard to measure model performance
accuracy=Proportion of correct predictions
precision=Proportion of true positives among predicted positives
recall=Proportion of true positives among actual positives
f1 score=Harmonic mean of precision and recall
confusion matrix=Table summarizing classification outcomes
roc curve=Graph showing tradeoff between true and false positive rates
auc=Area under the roc curve measuring performance
linear regression=Algorithm modeling linear relationship between variables
logistic regression=Algorithm for binary classification using sigmoid
knn=Algorithm classifying based on nearest neighbors
decision tree=Tree structured model splitting data by features
random forest=Ensemble of decision trees for improved accuracy
gradient boosting=Sequential ensemble improving errors of prior models
svm=Algorithm maximizing margin between classes
naive bayes=Probabilistic classifier based on bayes theorem
k means=Clustering algorithm partitioning data into k groups
hierarchical clustering=Clustering building nested group structure
pca=Dimensionality reduction using principal components
neural network=Model composed of interconnected neurons
perceptron=Single layer linear classifier
activation function=Nonlinear function applied to neuron output
backpropagation=Algorithm computing gradients for neural networks
deep learning=Machine learning using multiple neural network layers
convolutional network=Neural network specialized for images
recurrent network=Neural network for sequential data
transformer=Attention based architecture for sequence modeling
embedding=Vector representation of categorical data
natural language processing=Machine learning applied to human language
computer vision=Machine learning applied to image understanding
time series=Data collected over time intervals
anomaly detection=Identifying rare unusual data patterns
recommendation system=Model suggesting items to users
ensemble learning=Combining multiple models for better results
bagging=Ensemble method reducing variance
boosting=Ensemble method reducing bias
stacking=Ensemble method combining model outputs
pipeline=Structured sequence of data processing steps
deployment=Integrating trained model into production environment
monitoring=Tracking model performance after deployment
drift=Change in data distribution over time
explainability=Ability to interpret model predictions
